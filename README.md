## 前言
仿大众点评项目是本人毕业设计的一个电商系统，采用现阶段主流技术实现。

## 项目介绍
仿大众点评项目是一套电商系统，包括前台商城系统，基于SpringBoot+MyBatis实现，采用Docker容器化部署。前台商城系统包含首页门户、商品推荐、商品搜索、商品展示、购物车、订单流程等模块
